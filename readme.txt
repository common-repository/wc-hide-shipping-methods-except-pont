== Description ==
This plugin automatically hides all other shipping methods when “free shipping” is available during checkout process. It also includes an option to keep “local pickup” and “pont” (by Szathmari and by Viszt Péter) available alongside “free shipping”. Your donation is greatly appreciated.

== Installation ==
1. Download the plugin & install it to your `wp-content/plugins` folder (or use the Plugins menu through the WordPress Administration section)
2. Activate the plugin
3. Navigate to **WooCommerce > Settings > Shipping > Shipping options**.
4. Select your option.
5. Save and enjoy!
